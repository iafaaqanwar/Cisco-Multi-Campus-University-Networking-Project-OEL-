# Cisco-Multi-Campus-University-Networking-Project-OEL-
Open Ended Lab (OEL) project for Computer Networks (CSC-203L) simulating a scalable multi-campus university network (Main Campus + two Sub-Campuses) in Cisco Packet Tracer. Implements hierarchical network design, inter-VLAN routing, OSPF and BGP, server services (DNS, Web, FTP), and ACL-based security.
